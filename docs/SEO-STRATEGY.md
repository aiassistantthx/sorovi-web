## 4. On-Page SEO Specifications

### 4.1 Meta Tags Template

**Homepage:**
```html
<title>Sorovi - AI Video Generator App | Create TikTok & Reels from Photos</title>
<meta name="description" content="Transform your photos into stunning AI videos for TikTok, Instagram Reels, and YouTube Shorts. Zoom, orbit, and follow effects. Free download.">
```

**Tool Page:**
```html
<title>{Tool Name} | Sorovi AI Video Generator</title>
<meta name="description" content="Create {tool description} with Sorovi AI. Transform photos into {platform} videos in seconds. Download free for iOS.">
```

### 4.2 Header Hierarchy

```
H1: Main keyword (unique per page)
  H2: Feature sections
    H3: Individual features
  H2: How it works
    H3: Step 1, 2, 3
  H2: Use cases
    H3: Individual use cases
  H2: FAQ
    H3: Individual questions
```
